# 🚀 Automação de Organização de Arquivos

Script inteligente desenvolvido em Python para otimizar fluxos de trabalho, eliminando a bagunça de pastas com centenas de arquivos (como a pasta de Downloads ou relatórios de faturamento).

## 🛠️ O que esta solução faz:
* **Varredura Automática:** Lê todos os arquivos de uma pasta selecionada em segundos.
* **Classificação Inteligente:** Identifica a extensão do arquivo e o move para a pasta correta:
  * 📊 **Planilhas:** `.xlsx`, `.xls`, `.csv`
  * 📄 **PDFs:** `.pdf`
  * 📝 **Textos:** `.docx`, `.doc`, `.txt`
  * 🖼️ **Imagens:** `.jpg`, `.jpeg`, `.png`, `.gif`
* **Limpeza e Organização:** Cria subpastas automaticamente se elas não existirem, deixando o ambiente de trabalho limpo.

## 💻 Tecnologias Utilizadas:
* **Python 3** (Bibliotecas nativas `os` e `shutil` — não requer instalações externas).
